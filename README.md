## Projetos em destaque

## SISEP — Sistema Integrado de Escolas Profissionais  
Sistema para gerenciamento de processos seletivos com cotas, notas e classificação automática de candidatos.

**Minha contribuição:**
- Desenvolvimento da lógica de classificação dos alunos  
- Manipulação e organização de dados  
- Regras de desempate e critérios de seleção  
- Atuação no back-end do sistema  

Tecnologias: PHP, MySQL, Python (dados), Bootstrap  

Repositório: https://github.com/Muril08/SISEP

##  Conecta Local  
Aplicativo que conecta pequenos comércios locais a clientes, substituindo panfletos impressos por uma plataforma digital.

**Minha contribuição:**
- Desenvolvimento de funcionalidades de back-end  
- Estruturação de dados dos comércios e ofertas  
- Integração com banco de dados  

Tecnologias: JavaScript, Firebase, React Native  

Repositório: https://github.com/Muril08/ConectaLocal

## Piano Tutor  
Projeto educacional com Arduino que simula um piano para auxiliar iniciantes no aprendizado musical.

**Minha contribuição:**
- Modelagem 3D do piano  
- Auxílio no desenvolvimento do circuito com Arduino  
- Integração entre hardware e lógica do projeto  

Tecnologias: Arduino (C++), sensores, Tinkercad  

Repositório: https://github.com/Muril08/Piano-Tutor

## EcoCycle  
Projeto de vermicompostagem automatizada com Arduino, voltado ao monitoramento ambiental e educação sustentável.

**Minha contribuição:**
- Desenvolvimento do circuito com sensores  
- Programação em Arduino  
- Organização e estruturação dos dados coletados  

Repositório: https://github.com/Muril08/eco-cycle  
Site: https://eco-cycle-live-xclx.vercel.app/
